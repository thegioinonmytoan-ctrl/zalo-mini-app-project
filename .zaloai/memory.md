# BỘ NHỚ DỰ ÁN (PROJECT MEMORY)
Quy tắc chung cho AI khi làm việc với dự án này:

1. **Tech Stack:**
   - Framework: React (Zalo Mini App SDK)
   - UI Library: zmp-ui, TailwindCSS
   - State Management: Recoil

2. **Quy tắc Code:**
   - Luôn sử dụng TypeScript.
   - Không được hardcode màu sắc, hãy dùng class của Tailwind (ví dụ: text-blue-500).
   - Khi sửa lỗi, hãy giải thích nguyên nhân trước khi đưa ra code mới.

3. **Ghi chú Nghiệp vụ (Bạn hãy tự thêm vào đây):**
   - Ví dụ: Ứng dụng này là shop bán quần áo.
   - Ví dụ: Màu chủ đạo là Xanh Dương (#007aff).
